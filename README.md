# NRI Campus Dairy

## Features
- Email & Password Authentication
- Share Posts with Caption
- Display Posts with Caption
- Like & Comment on Posts
- Search Users
- Follow Users
- Display User Posts, Followers & Following
- EVERYTHING REAL TIME
- Sign Out

## Tech Stack

**Server**: 
- Firebase Auth, 
- Firebase Storage,
- Firebase Firestore

**Client**:
- Flutter,
- Provider
